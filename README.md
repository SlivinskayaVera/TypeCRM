# TypeCRM

**CRM-система с ролевой моделью на TypeScript + React**

Учебный проект для курса по TypeScript. Демонстрирует применение всех ключевых возможностей TypeScript в реальном React-приложении: от базовых типов до продвинутых паттернов.

---

## 🚀 Технологии

- **React 19** — современный UI-фреймворк
- **TypeScript** — 100% типобезопасный код
- **Vite** — быстрая сборка и разработка
- **React Router v7** — навигация и защита маршрутов
- **Zod** — валидация форм с автовыводом типов

---

## 📦 Функциональность

**👤 Для всех пользователей**
- Регистрация и вход
- Редактирование профиля (имя, фамилия, аватар)
- Смена пароля
- Персональный дашборд

**👑 Для администраторов**
- Управление пользователями
- Изменение ролей
- Удаление пользователей

---

## 🎯 Цели проекта

Этот проект создан для демонстрации TypeScript в действии:

- Интерфейсы и union типы
- Дженерики в компонентах и хуках
- Discriminated unions для состояний
- Type guards и сужение типов
- Utility types (`Partial`, `Omit`, `Record`)
- Zod + TypeScript (`z.infer`)
- TypeScript + React Context
- Типизация API-клиента
